# TypeVault Website

Free static landing page for TypeVault.

## Files
- `index.html` - main website
- `style.css` - website styling
- `script.js` - scroll animation and download placeholder alert
- `version.json` - update checker file for the TypeVault app
- `assets/` - logo and screenshots

## GitHub Pages setup
1. Create a public GitHub repository named `typevault`.
2. Upload all files from this folder.
3. Go to Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Save.

Your site will become available at:
`https://YOURUSERNAME.github.io/typevault/`

Your update file will be:
`https://YOURUSERNAME.github.io/typevault/version.json`

## Important
Replace the placeholder download button link in `index.html` with your real GitHub Release EXE/ZIP link.
Also update `version.json` when a newer version is released.
